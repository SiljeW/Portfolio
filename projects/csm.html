    Project Overview

    The Community Science Museum website contains a presentation of what there is to explore at the museum.

    The core target audience is primary and middle school children (ages 7-15) and families with young children, with a purpose to excite and entice people to visit the museum

    This website was our first semester project in 2023, to demonstrate the ability to create user friendly design in Figma, write semantic HTML code and style with CSS

    Key Features

    • Responsive design for all device sizes
    • Cards to highlight exhibits and main events
    • Contact form and visitor information
