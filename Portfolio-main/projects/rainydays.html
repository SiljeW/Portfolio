    Project Overview

    Rainy Days is an e-commerce site specializing in high-quality outdoor jackets for men and women. The goal was to create a responsive online shop with a clean, modern design that showcases the products effectively while providing an intuitive shopping experience.

    The target audience includes outdoor enthusiasts aged 30-50 who prioritize quality, durability, and functionality in their outdoor gear. The brand positioning emphasizes "Pushing the Comfort Zone" with jackets designed for various outdoor activities.

    This project was the first Figma design course, HTML and CSS course in 2023, and was later brought up in the first Javascript course in 2024.

    Key Features

    • Responsive design for all device sizes
    • Product filtering by category, price, and activity type
    • Detailed product pages with size guide and specifications
    • Streamlined checkout process
    • User account management
